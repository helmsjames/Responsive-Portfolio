# Responsive-Portfolio

This is a rough work-in-progress "about me" series of pages.
You should be able to select a link. For example, the About link should take you to a page that gives you a breif background about me. Selecting the Contact link brings up another page that you can use to send me a message. And, the Portfolio link opens up a page that provides images which are snapshots or thumbnails related to projects I have worked on.
